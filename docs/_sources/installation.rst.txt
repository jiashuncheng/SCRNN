Installation
============

Clone the repository using either of the following two commands:

.. code-block:: bash

    $ git clone https://github.com/BasBuller/PySNN.git
    $ git clone git@github.com:BasBuller/PySNN.git

Installation can be done with pip:

.. code-block:: bash

    $ pip install PySNN/

If you want to make updates to the library without having to reinstall it, use the following install command instead:

.. code-block:: bash

    $ pip install -e PySNN/

Code is formatted with `Black <https://github.com/psf/black>`_ using a pre-commit hook. To configure it, run:

.. code-block:: bash

    $ pre-commit install
